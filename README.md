### 📋 Catálogo Unidas Medical · QR Code + Lead Capture
 
Sistema completo de captação de leads via **QR Code**, com formulário responsivo e integração ao **Supabase** em tempo real.
 
**🎯 O que faz:**
- 📱 QR Code direciona ao formulário de cadastro
- 📝 Captura dados de leads (nome, empresa, telefone, e-mail, CNPJ)
- 💾 Armazenamento em tempo real no **Supabase**
- 🔗 Redireciona ao catálogo após preenchimento
**🛠️ Tecnologias:** `HTML5` `CSS3` `JavaScript` `Supabase` `Netlify` `QRCode.js`
 
🌐 **Demo:** [catalogounidas.netlify.app](https://catalogounidas.netlify.app)
